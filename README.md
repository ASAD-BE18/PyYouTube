# PyYouTube 

Get Video Data from YouTube link 

## Installation 
```pip install PyYouTube```

## How to use it ?


```python
from pyyoutube import Data
yt = Data("https://youtu.be/HhHzCfrqsoE")
print(yt.data)
```
